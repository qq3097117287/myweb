# 智慧图书管理系统 - 使用说明

## 快速开始

### 1. 数据库设置

#### 方法一：使用初始化脚本（推荐）
```bash
# 登录MySQL
mysql -u root -p

# 执行初始化脚本
source C:/Users/Administrator/Desktop/library/init_db.sql
```

#### 方法二：手动创建
1. 创建数据库 `db_library`
2. 运行 `init_db.sql` 中的建表语句
3. 插入示例数据

### 2. 配置数据库连接

编辑 `app.js` 文件，修改数据库连接配置：
```javascript
const db = mysql.createConnection({
    host: 'localhost',
    user: 'root',        // 修改为你的MySQL用户名
    password: 'root',    // 修改为你的MySQL密码
    database: 'db_library'
})
```

### 3. 安装依赖并运行

```bash
# 进入项目目录
cd C:\Users\Administrator\Desktop\library

# 安装依赖
npm install

# 运行项目（开发模式）
npm run dev

# 或者运行项目（生产模式）
npm start
```

### 4. 访问系统

打开浏览器访问：http://localhost:3000/login

## 功能使用指南

### 管理员功能

#### 1. 数据概览（首页）
- 查看图书总数、读者总数、在借图书、已归还图书
- 查看数据图表（图书分类分布、借阅状态统计）
- 快捷操作入口

#### 2. 图书管理
- **查看图书**：浏览所有图书信息
- **搜索图书**：按书名、作者、ISBN搜索
- **筛选图书**：按分类筛选
- **排序图书**：按书名、作者排序
- **添加图书**：点击"新增图书"按钮
- **编辑图书**：点击编辑图标修改图书信息
- **删除图书**：点击删除图标删除图书

#### 3. 读者管理
- **查看读者**：浏览所有读者信息
- **搜索读者**：按姓名、学号、电话搜索
- **添加读者**：点击"新增读者"按钮
- **编辑读者**：点击编辑图标修改读者信息
- **删除读者**：点击删除图标删除读者

#### 4. 借阅管理
- **查看借阅记录**：浏览所有借阅记录
- **归还图书**：点击"归还"按钮完成归还操作
- **搜索借阅**：按图书名称或借阅人搜索

### 读者功能

#### 1. 图书阅览
- 浏览所有可借阅图书
- 查看图书详细信息（书名、作者、分类）
- 查看借阅状态（可借阅/已被借出）
- 点击"借阅"按钮借阅图书

#### 2. 我的借阅
- 查看个人借阅记录
- 查看借阅日期、归还日期
- 查看借阅状态（借阅中/已归还）

## 数据库表结构

### 图书表 (book)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| book_name | VARCHAR(255) | 图书名称 |
| author | VARCHAR(100) | 作者 |
| isbn | VARCHAR(20) | ISBN号 |
| category | VARCHAR(50) | 分类 |
| stock | INT | 库存数量 |
| description | TEXT | 简介 |
| created_at | TIMESTAMP | 创建时间 |

### 读者表 (reader)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| read_name | VARCHAR(100) | 读者姓名 |
| student_id | VARCHAR(20) | 学号/工号 |
| phone | VARCHAR(20) | 电话 |
| email | VARCHAR(100) | 邮箱 |
| created_at | TIMESTAMP | 创建时间 |

### 借阅记录表 (borrow)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| book_id | INT | 图书ID（外键） |
| reader_id | INT | 读者ID（外键） |
| borrow_date | DATE | 借阅日期 |
| back_date | DATE | 归还日期 |
| created_at | TIMESTAMP | 创建时间 |

### 用户表 (sys_user)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT | 主键 |
| username | VARCHAR(50) | 用户名 |
| password | VARCHAR(100) | 密码 |
| role | VARCHAR(20) | 角色（admin/reader） |
| created_at | TIMESTAMP | 创建时间 |

## 常见问题

### Q1: 数据库连接失败
**解决方法**：
1. 检查MySQL服务是否启动
2. 检查数据库连接配置（用户名、密码、数据库名）
3. 确保数据库 `db_library` 已创建

### Q2: 端口被占用
**解决方法**：
修改 `app.js` 中的端口号：
```javascript
const port = 3001  // 改为其他端口
```

### Q3: 无法访问页面
**解决方法**：
1. 确保项目已启动（查看终端输出）
2. 检查防火墙设置
3. 尝试使用 `http://127.0.0.1:3000/login`
